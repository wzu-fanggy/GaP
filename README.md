# Crystal Structure and Property Prediction of Gallium Phosphide via Generative Adversarial Network

本仓库提供 **GAN 模型代码** 与 **GaP 体系晶体结构数据集（CIF）**。  
数据集基于 [Materials Project (MP)](https://materialsproject.org/) 数据库进行扩增构建，并在本仓库中以可下载、可访问的形式发布。
---
## 1. 仓库内容概览
  
### (A) GAN 项目代码
 
  本仓库提供使用的GAN模型代码。

### (B) GaP 数据集 
   来自 MP 数据库的 **8 个 GaP 初始结构**（用于扩增起点/基准）。以及共 **9974** 个扩增得到的晶体结构样本， 选取的**8 个代表性结构**
